## 🚀 Примеры использования

### Макросы
```
1. Создать макрос "Открыть браузер":
   - Клик (100, 50) — кнопка Пуск
   - Задержка 0.5с
   - Текст "chrome"
   - Клавиша "enter"

2. Записать макрос работы в программе:
   - Запись 30 сек
   - Выполнить действия
   - Сохранить как "Обработка данных"
   - Запустить 5x
```

### Планировщик
```
1. Ежедневный бэкап:
   - Время: 02:00
   - Команда: xcopy C:\Data D:\Backup /E /Y
   - Повторять: ежедневно

2. Проверка обновлений:
   - Время: 09:00
   - Команда: winget upgrade --all
   - Повторять: ежедневно
```

### Установка программ
```
1. Через Winget:
   - Поиск: "chrome"
   - Установка: Google.Chrome

2. По URL:
   - https://example.com/setup.exe
   - Автоматическая установка

3. Из файла:
   - Отправить .exe в чат
   - Автоматическая установка
```

## 📝 Технические детали

### Зависимости
Все функции используют стандартные библиотеки, кроме:
- `pyautogui` — для макросов (уже установлен)
- `requests` — для установки по URL (опционально)

### Файлы данных
```
terminal_history.json — история команд терминала
scheduled_tasks.json — запланированные задачи
macros.json — сохранённые макросы
```

### Безопасность
- Все функции проверяют `is_allowed()`
- Персональные данные для каждого пользователя
- Логирование всех операций
- Таймауты для длительных операций
