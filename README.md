## `> QUICK.BOOT`

### Требования

```
Runtime   ─── Python 3.8+
Platform  ─── Windows 10/11
Account   ─── Telegram Bot Token (@BotFather)
Optional  ─── AnyDesk (для GUI доступа)
```

### `// STEP 1 — клонирование`

```bash
git clone https://github.com/Bifyp/BifypPC
cd BifypPC
```

### `// STEP 2 — зависимости`

```bash
pip install -r requirements.txt
```

### `// STEP 3 — конфигурация`

```bash
cp .env.example .env
```

Заполните `.env`:

```env
# ── BOT ───────────────────────────────────────────
BOT_TOKEN=your_bot_token_here
ALLOWED_USER_IDS=123456789,987654321

# ── SECURITY ──────────────────────────────────────
WEB_PANEL_PASSWORD=YourSecureP@ssw0rd!2024
SESSION_LIFETIME_HOURS=2
MAX_LOGIN_ATTEMPTS=5
NOTIFY_ON_WEB_LOGIN=True

# ── ALERTS ────────────────────────────────────────
ALERT_CPU_THRESHOLD=80
ALERT_RAM_THRESHOLD=85
ALERT_DISK_THRESHOLD=90
```

> Получить токен — `@BotFather` · Узнать свой ID — `@userinfobot`

### `// STEP 4 — запуск`

```bash
# Через Python
python bot.py

# Через bat-файл (Windows)
start_bot.bat

# С watchdog (автоперезапуск)
start_watchdog.vbs
```
